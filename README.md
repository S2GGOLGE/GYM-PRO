# 🏋️ GYM-PRO

GYM-PRO, spor salonları için geliştirilmiş kapsamlı bir yönetim ve üyelik takip sistemidir.  
Kullanıcıların hesap oluşturabildiği, giriş yapabildiği ve abonelik süreçlerini yönetebildiği;  
admin tarafında ise sistemin merkezi olarak kontrol edilebildiği bir yapıya sahiptir.

---

## 🔧 Özellikler

- Kullanıcı kayıt & giriş sistemi (hash + token altyapısı)
- Profil yönetimi
- Üyelik / abonelik yönetimi
- Admin paneli (geliştirme aşamasında)
- Modal tabanlı modern UI sistemi
- Güvenli şifreleme (hashleme)

---

## 🛠 Kullanılan Teknolojiler

- ASP.NET Core
- C#
- SQL Server
- JavaScript
- HTML / CSS

## 🎯 Amaç

Bu proje, modern yazılım geliştirme prensipleri kullanılarak gerçek dünya senaryolarına uygun,  
ölçeklenebilir ve güvenli bir spor salonu yönetim sistemi oluşturmayı hedeflemektedir.

Aynı zamanda:
- Backend & frontend entegrasyonu
- Güvenlik (authentication / authorization)
- Veritabanı yönetimi

gibi kritik konularda pratik bir geliştirme deneyimi sunar.
